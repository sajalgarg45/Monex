# Finance Assistant

A comprehensive finance tracking app built with SwiftUI for iOS.

## Features

- **User Authentication**: Simple login interface
- **Dashboard**: Overview of all finances with summary cards and pie charts
- **Budget Management**: Create, edit, and delete budget categories
- **Expense Tracking**: Add expenses to specific budget categories
- **Visual Analytics**: Charts to visualize spending patterns
- **Data Persistence**: Local storage for budget and expense data

## Requirements

- iOS 15.0+ (Charts feature requires iOS 16.0+)
- Xcode 14.0+
- Swift 5.0+

## Architecture

The app follows a clean architecture with:

- **Models**: Data structures for User, Budget, and Expense
- **ViewModels**: Business logic and data management
- **Views**: UI components built with SwiftUI

## Getting Started

1. Clone the repository
2. Open the project in Xcode
3. Build and run the app
4. Login using any email and password (mock authentication)
5. Start adding budgets and tracking expenses

## Design Principles

This app adheres to Apple's Human Interface Guidelines and implements the following design principles:

- **Minimum touch targets of 44x44 points**
- **Readable text sizes (11pt minimum)**
- **Clear visual hierarchy**
- **Appropriate spacing and alignment**
- **High-contrast color schemes**
- **Intuitive navigation**
